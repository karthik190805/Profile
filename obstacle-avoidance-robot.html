Obstacle Avoiding Robot

I have completed my project on building an autonomous obstacle-avoiding robot! This project was a fantastic learning experience in robotics fundamentals, sensor integration, programming with Arduino, etc.. The robot uses an ultrasonic sensor to detect obstacles and navigate its path accordingly. I encountered challenges in Powering the Arduino and Making the robot to move, but overcame them by adding a Battery Bank which provided enough amount of Power required by the robot. This project has solidified my interest in robotics, automation, AI.

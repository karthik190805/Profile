Password Based Door Lock System

Imagine a smart door lock that requires a password to open. Instead of a physical key, you enter a secret code. How Does It Work? We use an Arduino (a tiny computer) to control the system. The system includes:

  • Keypad: Where you enter the password.
  • Electromagnetic Lock: The lock that opens and closes.
  • Relay: Connects the Arduino to the lock.
  • LCD (optional): Displays messages (like “Door Unlocked”).
When you enter the correct password, the lock opens!
